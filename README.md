# Pulsar Detection Using Machine Learning

## Overview
This project uses machine learning techniques to classify pulsar candidates using the HTRU2 dataset.

## Techniques Used
- Python
- Scikit-learn
- Decision Trees
- Cross-validation
- Feature selection

## Features Used
- Mean_Integrated
- Excess_Kurtosis_Integrated
- Skewness_Integrated
- Mean_DM_SNR
- SD_DM_SNR

## Results
The model was evaluated using confusion matrices, learning curves, and cross-validation.
